# Game Theory Comparison

A simple interactive web app to compare classic game theory models side by side.

## Overview
This website allows users to select two game theory scenarios and explore how they differ in structure, incentives, and outcomes. It is designed for intuitive learning and reflection rather than heavy theory.

## Features
- Compare two game theory models simultaneously  
- Clean, minimal UI with responsive design  
- Real-world examples for each game  
- Clear explanations of:
  - Why the game occurs  
  - How it works  
  - Core strategic insight  
- Reflection prompts to connect concepts to real-life situations  
- Automatic comparison section showing:
  - Common themes  
  - Key differences  

## Included Game Theory Models
- Prisoner's Dilemma  
- Stag Hunt  
- Chicken  
- Ultimatum Game  
- Vickrey Auction  
- Centipede Game  
- Battle of the Sexes  
- Public Goods Game  
- Tragedy of the Commons  
- Repeated Prisoner's Dilemma  
- Signaling Game  
- Nash Bargaining  
- Stackelberg Competition  
- Matching Pennies  

## Tech Stack
- HTML  
- CSS (custom styling, responsive layout)  
- Vanilla JavaScript  

## How to Use
1. Visit the website: <a href="https://sanjaykohli.github.io/GT-Analysis/" target="_blank">Game Theory Comparison</a>  
2. Select two games from the dropdown menus  
3. Read and compare the insights, examples, and strategies  

## Purpose
This project helps users build intuition about strategic decision-making, incentives, and human behavior through interactive comparison.
